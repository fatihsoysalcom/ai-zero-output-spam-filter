# AI Zero Output Spam Filter

This example demonstrates the 'Zero Output Paradigm' for AI agents. A `ZeroOutputSpamFilterAgent` silently processes an inbox, removing spam without notifying the user about its actions or the number of emails filtered. The value is delivered through the absence of noise, providing a clean and efficient user experience without explicit interaction.

## Language

`python`

## How to Run

1. Save the code as `main.py`.
2. Run from your terminal: `python main.py`
